Integrate your Meetup.com group with your website.

== Description ==

Meetup Connect is a powerful, flexible platform for integrating with meetup.com. It aims to provide full feature integration in a simple, easy to use package allowing you to seamlessly integrate Meetup with your WordPress site.

**Follow this plugin on [GitHub](https://github.com/Section214/Meetup-Connect)**

== Installation ==

1. Activate the plugin
2. Go to Meetup Connect > Settings and configure the plugin
3. Add shortcodes and/or widgets to your website
4. Enjoy!
